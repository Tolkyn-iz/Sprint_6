# Sprint_6 - Автотесты для сервиса "Яндекс.Самокат"

## Описание проекта
Автотесты для тестирования функциональности сервиса "Яндекс.Самокат" (https://qa-scooter.praktikum-services.ru/).

## Технологии
- Python 3.14+
- Selenium
- Pytest
- Allure
- Page Object Model

## Тестовые сценарии

### 1. Выпадающий список в разделе "Вопросы о важном"
- 8 тестов на проверку открытия текста при нажатии на стрелочку

### 2. Заказ самоката
- Позитивный сценарий с двумя наборами данных
- Проверка двух точек входа (кнопка "Заказать" вверху и внизу)

### 3. Проверка переходов по логотипам
- Переход на главную страницу при клике на логотип "Самоката"
- Открытие "Дзена" в новом окне при клике на логотип Яндекса
